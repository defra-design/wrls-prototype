Help to enter multiple volumes or readings into a return

You can enter multiple values into a return by copying and pasting from a document on your device. For example, from an Excel spreadsheet or a text file.

If you're using a spreadsheet, each of the values should be in one column. They should be in date order.

Select and then copy the values column. You do not need the dates.

Paste the values into the box on the form. The values will appear in a column.

For this form to work, values should:

  • either be on separate lines or be separated by a comma
  • cover the entire return period - if a value is missing, use an x to fill the gap
  • run sequentially to match the return period dates
