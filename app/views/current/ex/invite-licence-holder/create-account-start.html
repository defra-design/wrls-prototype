Create an account to manage your water abstraction licence online

You must be a licence holder to create an online account. Once registered you will be able to:

  • view your abstraction licence information
  • submit abstraction returns
  • give others access to your account and permission to submit returns

To protect your data we have to post a security code to the registered address of the licence.

We will send a security code to you straight away, using first-class mail.

Impoundment licences are not yet part of this service.

Before you start

You will need:

  • an email address
  • your licence numbers
  • the code we emailed you
