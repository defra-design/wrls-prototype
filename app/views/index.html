Manually adding a licence to a supplementary bill run - 13 September 2021

When a licence has a TPT agreement, B&D may need to specify that it needs to be supplementary billed. This is so they can re-calculate the amount of water that is used in the second part charge on a TpT.

The scenario can occur if someone submitted returns late and there was a legitimate reason for rebilling them. This process has three parts, marking the licence for supplementary billing, reviewing the changes to the billable volumes and sending the revised bills. The sending part of the process in the prototype is the same as supplementary billing. B&D will need to mark up the downloaded bill run with specific actions for SSCL.

The user can mark the licence for the supplementary bill by following the flow form the "recalculate bills" action on the TpT agreement.

  • TpT agreement with "recalculate bills" action
  • You're about to recalculate bills
  • You've marked this licence for the next supplementary bill run
  • Licence marked for supplementary billing
